# BurnIN Test Configuration

This directory contains BurnIN test configuration files:

- `BurnInScript.bitcfg` - BurnIN test configuration (binary format)
- `BurnInScript.bits` - BurnIN test script (auto-generated by test)

## Notes

- The `.bits` file is automatically generated during test execution
- The `.bitcfg` file is copied from the original STC-625 test case
- Do not manually edit these files unless necessary
